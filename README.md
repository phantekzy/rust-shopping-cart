# Rust Shopping Cart

A command-line interface (CLI) application built in Rust that calculates a 
shopping receipt based on user input for items, prices, and quantities.

## Features

- Dynamic string manipulation for pluralization.
- Type-safe input parsing using Rust's standard library.
- Formatted currency output.
- Memory-efficient input handling with string clearing.

## Requirements

- Rust (edition 2021 or later)
- Cargo

## How to Run

1. Clone the repository:
   git clone https://github.com/phantekzy/rust-shopping-cart.git

2. Navigate into the directory:
   cd rust-shopping-cart

3. Build and run the project:
   cargo run

## Implementation Details

The project demonstrates core Rust concepts including:
- Ownership and Borrowing.
- String vs. &str.
- Type Inference.
- Error handling with expect().
- Standard I/O flushing.
